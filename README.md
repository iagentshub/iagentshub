<h1 align="center">iAgentsHub</h1>

<p align="center">AI agents platform. One command deploys the full stack.</p>

---

## Quick deploy

```bash
git clone https://github.com/iagentshub/iagentshub.git
cd iagentshub
cp .env.example .env
# Edit .env — set GAIA_AGENTS_SECRET
./gaia.sh start
```

Open `http://localhost`.
